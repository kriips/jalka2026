# Country flags

SVG flags used for team badges (group/playoff fixtures, favourite-team
badges on the leaderboard, user profiles, etc.). Filenames are ISO 3166-1
alpha-2 codes, plus the FIFA sub-regions `gb-eng` (England) and `gb-sct`
(Scotland). The paths are referenced from the `teams.flag` column — see
`Jalka2026.Seed.Parser` and the `*_team_flags_to_local` migrations.
